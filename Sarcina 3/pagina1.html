Introducere în HTML

Pagina 2 | Pagina 3

HTML (HyperText Markup Language) este limbajul standard utilizat pentru crearea paginilor web. Acesta permite structurarea conținutului prin elemente precum titluri, paragrafe, liste și linkuri.

Fiecare pagină web este alcătuită dintr-o structură de bază care include secțiuni precum head și body. În secțiunea head sunt incluse informații despre pagină, iar în body se află conținutul vizibil.

Utilizarea corectă a elementelor HTML contribuie la organizarea clară a informației și la o experiență mai bună pentru utilizatori.

Elemente HTML de bază

Element Descriere Exemplu
<h1> Titlu principal <h1>Titlu</h1>
<p> Paragraf <p>Text</p>
<a> Link <a href="#">Link</a>
